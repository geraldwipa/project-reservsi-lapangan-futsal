Berikut Adalah Cara untuk Instal Aplikasi ini:
1. Instal dulu Xampp v 3.2.2
2. pastikan mysql dan apache telah aktif
3. Exktract aplikasi Futsal kemudian ada folder "Crud_ci".
4. Copy folder Crud_ci pada :C/Xampp/htdocts/
5. buatlah database dengan nama "mynotescode". kemudian
Import tabel pada file "mynotescode.sql" pada aplikasi ini
6. jalankan Aplikasi ini dengan memanggil https: localhost/crud_ci
7. masukan username "admin" dan
   masukan password "admin"
8. selesai
